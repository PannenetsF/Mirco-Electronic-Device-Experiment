\section{实验目的}

测量二极管的直流特性并进行分析，并从中：

\begin{itemize}
    \item 了解通用仪表的的基本原理和使用方法
    \item 了解被测器件各项参数的定义和测量方法
    \item 掌握被测器件直流特性和相关机理
\end{itemize}

\section{实验所用设备及器件}

实验用到的核心器件是二极管，其他设备包括面包板、杜邦线、电压源、手持万用表、台式万用表等。

\section{实验基本原理及步骤}

基本原理是基于二极管的单向导通特性连接电路后，在不同电压下读取对应的电流，来获得对应的直流特性。
